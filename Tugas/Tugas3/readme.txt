Hi! Selamat datang di program untuk menghitung akar dari sebuah fungsi
Untuk program ini fungsi yang digunakan adalah f(x)= 5-5x-e^(0.5*x)

Ikut langkah-langkah di bawah ini untuk menjalankan program:
1. Jalankan program menggunakan python3 atau versi setelahnya
2. Setelah program berhasil dijalankan, silahkan pilih jenis metode yang ingin kamu gunakan.
3. Jika ingin menggunakan metode Bisection, ketik 1 lalu enter.
    a. Kemudian kamu akan diminta untuk memasukkan nilai tebakan batas bawah dan batas atas, serta approximate error rate.
    b. Masukkanlah nilai sesuai dengan yang diminta.
    c. Enter setiap kamu selesai memasukkan nilai.
    d. Program selanjutkan akan menghitung menggunakan metode Bisection.
    e. Hasil akan ditampilkan.
    f. Silahkan memilih dari menu apa yang akan kamu lakukan selanjutnya.
4. Jika ingin menggunakan metode Newton Raphson, ketik 2 lalu enter.
    a. Kemudian kamu akan diminta untuk memasukkan nilai tebakan untuk xi dan approximate error rate.
    b. Masukkanlah nilai sesuai dengan yang diminta.
    c. Enter setiap kamu selesai memasukkan nilai.
    d. Program selanjutkan akan menghitung menggunakan metode NewtonRaphson.
    e. Hasil akan ditampilkan.
    f. Silahkan memilih dari menu apa yang akan kamu lakukan selanjutnya.
5. Jika ingin menggunakan metode Secant, ketik 3 lalu enter.
    a. Kemudian kamu akan diminta untuk memasukkan nilai tebakan untuk xi-1 dan xi, serta approximate error rate.
    b. Masukkanlah nilai sesuai dengan yang diminta.
    c. Enter setiap kamu selesai memasukkan nilai.
    d. Program selanjutkan akan menghitung menggunakan metode NewtonRaphson.
    e. Hasil akan ditampilkan.
    f. Silahkan memilih dari menu apa yang akan kamu lakukan selanjutnya.
6. Silahkan memilih di antara angka 1-3 kembali jika kamu ingin lanjut menghitung.
7. Jika kamu telah selesai, masukkan angka 4.
8. Program akan selesai.
9. Terima kasih telah menggunakan program
